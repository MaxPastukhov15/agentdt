# 📚 Desktop AI Assistant

Интеллектуальный десктопный ассистент, объединяющий локальную работу с вашими документами и мощь глобального поиска в сети. Программа позволяет извлекать знания из коллекции PDF-книг и дополнять их актуальной информацией из интернета или Arxiv в едином интерфейсе.

## ✨ Основные возможности

* **Локальный контекст (RAG):** Загружайте свои книги и документы в формате PDF в папку `pdf_docs`, и ассистент будет использовать их как основную базу знаний.
* **Web-поиск:** Если ответа нет в документах, агент автоматически найдет нужные статьи и информацию в сети.
* **Интуитивный интерфейс:** Удобное графическое приложение для управления диалогами.

## 🚀 Быстрый старт

### 1. Подготовка API Key
Для работы ассистента необходимо получить ключ доступа:
1. Перейдите на [OpenRouter](https://openrouter.ai/).
2. Создайте аккаунт и сгенерируйте новый **API Key**.
3. Сохраните его — он понадобится при первом запуске приложения.

### 2. Установка
```bash
# Клонируйте репозиторий
git clone https://github.com/your-username/desktop-ai-assistant.git

# Перейдите в директорию проекта
cd desktop-ai-assistant

# Установите необходимые зависимости
python -m venv venv

.../venv/Scripts/Activate.ps1

pip install -e .
```

### 3. Подготовка документов
Поместите все ваши PDF-файлы, по которым будет проводиться поиск, в папку:
`pdf_docs/`

### 4. Запуск
```bash
python main.py
```
